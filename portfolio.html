Google Action - You Choose Adventure

2020-09-30
Audio Animation
Released 8 voice and touch interactive actions for Google touch screen and audio devices. These interactive adventures include professionally recorded audio and limited animation. Users can make choices with voice recognition or on screen touch points.
For this project we were requested to convert 8 of Capstones you choose adventure Alexa skills into Google actions with added visuals and touch screen capabilities. Working with Earplay developers we learned how to add animation capabilities through the Earplay platform. A challenge we overcame was figuring out how to create loti animation files and integrate their json exports. Analytics have been implemented to track users choices throughout the action.

InDesign Export Scripts

2019-10-15
Javascript
These utility scripts were created to combine several export and file renaming processes for publishers preparing their products for print.
Users were having to perform several exports to prepare files for multiple formats. This included having to rename files and incerting cover pages. These utility scripts pull in users export presets and allows the user to specify multiple exports. Users can specify multiple ISBNs for files to be renamed to. User inputs are saved within the InDesign file for future use. Upon export files are renamed and mover into created directories in an agreed upon file structure.

Alexa Skills - You Choose Adventure

2017-12-15
Audio
Released 50 voice interactive skills for the Amazon Alexa device. Stories are professionally recorded and respond to users voice to make choices to advance the story.
I worked with a team to fulfill a request to convert 50 chose your own adventure titles into Alexa skills. The 50 titles selected were professionally recorded. Integrating the skills into Amazon's console required creating language models, designating choice paths, and setting up interaction variables to allow bookmarks, purchasing, and tracking which title the user has active.
